一个还不错的开源 QQ 机器人框架，基于 `Node.js` 和 `oicq2`。

`@kivibot/core` is the **core** library for `KiviBot`。

## Usage

Use `KiviBot CLI` to run `KiviBot`。

```shell
npm i -g kivibot --registry=https://registry.npmmirror.com

mkdir bot && cd bot

kivi init --start
```
